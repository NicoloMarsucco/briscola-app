# Briscola App

A simple iOS/Android implementation of the popular Italian card game [Briscola](https://en.wikipedia.org/wiki/Briscola), built with Flutter.

## Table of Contents

- [Features](#features)
- [Recordings](#recordings)
- [Getting Started](#getting-started)
- [Acknowledgments](#acknowledgments)

## Features

- **Multiple bot difficulty levels:**
    - **Easy:** plays randomly
    - **Medium:** uses simple heuristics
    - **Hard:** uses Monte Carlo Tree Search
- **GitHub-style game chart** to visualise game history and progress
- **Persistent settings** to control audio and visual preferences
- **Colourful and intuitive UI**, adaptive to various screen sizes on iOS and Android
- **Card animations** for movements and flipping
- **Unit-tested game logic** for reliability
- Uses **traditional Neapolitan playing cards**

`cd briscola_app`

`flutter pub get`

`flutter run`
